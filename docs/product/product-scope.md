# Product Scope

Energy Bill Lab helps U.S. residents understand electricity bills, estimate appliance costs, compare state electricity rates, and learn practical ways to reduce home-energy expenses.

## Current Status

The repository currently contains the monorepo foundation, web application shell, trust and legal pages, route registry, metadata helpers, sitemap, robots, and design-system tokens.

Calculator functionality is not implemented yet.

## Out Of Scope Until A Future Task

- Calculator formulas and result workflows
- Database schema and migrations
- EIA data ingestion
- Authentication and admin tools
- Advertising placement
- Analytics events
- User accounts or saved reports
